* nginx source code study

nginx源代码笔记。基于nginx版本1.16。

从nginx核心部分开始，对nginx的基础数据结构、流程、模块进行分析整理。

对于函数，在定义处才加以分析。

对于复杂的结构题的类型的定义直接进行分析，并适当给出使用方法和示例。

struct ngx_command_s {
